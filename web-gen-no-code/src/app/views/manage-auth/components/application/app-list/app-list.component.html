Chuyển đổi SQL script thành API

Điền truy vấn cơ sở dữ liệu của bạn tại đây để chuyển đổi thành API:

  • Ứng dụng chỉ hỗ trợ truy vấn PostgreSQL
  • Nếu cấu lệnh tạo cở sở dữ liệu hợp lệ thì các APIs tương ứng sẽ được tạo tại mục "APIs"
  • Nếu bạn muốn chuyển đỗi lại vui lòng thay đỗi truy vấn bên dưới và ấn nút "Chuyển đỗi lại" (Lưu ý: Dữ liệu và các API đã được tạo sẽ bị xoá)
